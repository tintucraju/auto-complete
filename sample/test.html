Tin Complete - A simple auto complete jquery plugin

Author : Tintu C Raju

http://www.tintu.co.vu

Local json Demo 


	Usage :

	var places = ['India','America','China','Japan','Korea','Nepal','Indonatia'];
	$("#myText").auto(
	{
			json : places,
			minLength:2,
			resultCount:4
	});
Remote Data source 

	Usage :
	
				
	$("#txtRemote").auto(
	{
		ajax : 'ajax.php',
		minLength:1,
		resultCount:3
	});
